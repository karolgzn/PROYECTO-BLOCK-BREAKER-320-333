# Música de Fondo

Coloca aquí tu archivo de música de fondo en formato compatible con SFML:
- `.ogg` (recomendado)
- `.wav`
- `.flac`
- `.mp3` (requiere dependencias adicionales)

## Archivo esperado:
- `background_music.ogg` - Música principal del juego
