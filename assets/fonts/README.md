# Fuentes

Para que el juego muestre el texto correctamente, copia una fuente TrueType (.ttf) en esta carpeta.

## Opción 1: Usar fuente del sistema (Recomendado)
El juego intentará usar automáticamente la fuente Arial de Windows si no encuentra una fuente aquí.

## Opción 2: Copiar manualmente
Puedes copiar cualquier fuente .ttf y renombrarla como `arial.ttf`

Ejemplos de ubicaciones en Windows:
- `C:\Windows\Fonts\arial.ttf`
- `C:\Windows\Fonts\calibri.ttf`
- `C:\Windows\Fonts\consola.ttf`

Simplemente copia el archivo a esta carpeta.
